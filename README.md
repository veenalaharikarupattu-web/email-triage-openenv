# Email Triage RL Environment
This is a Round 1 submission for the Scaler X Meta Hackathon.

## 📋 Overview
An Reinforcement Learning (RL) environment built using Meta's OpenEnv standards. 
It simulates a professional inbox where an AI agent must learn to prioritize 
urgent emails and categorize tasks efficiently.

## 🚀 Setup
- **SDK:** Docker
- **Framework:** FastAPI / Uvicorn
- **Environment Logic:** `env.py`
- **Scenarios:** `tasks.py` (Easy, Medium, Hard)
